OICM 使用 Markdown 渲染文本，在主论坛、私信、群组、博客等都是如此。若希望获得更丰富的展现，请使用 Markdown 和 KaTeX。

OICM 使用 Markdown-It 作为 Markdown 渲染引擎，并安装有以下插件：

  • markdown-it-katex - KaTeX 兼容
  • markdown-it-imsize - 图片大小调整
  • markdown-it-footnote - 脚注
  • markdown-it-highlightjs - 代码高亮
  • markdown-it-sub - 下标
  • markdown-it-sup - 上标
  • markdown-it-spoiler - 隐藏文字/吐槽
  • markdown-it-mark - 高亮
  • markdown-it-merge-cells - 列表扩展
  • markdown-it-media (改自 HydroOJ) - 媒体等的支持

注意：全部渲染插件均安装在服务端。Markdown 编辑器使用洛谷团队的 Markdown*Palettes，渲染结果可能和本网站不一致。

OICM 使用 KaTeX 作为 LaTeX 渲染引擎

常用扩展语法

==Text==Text
!!Text!!Text
^Sup^ ~Sub~SupSub
@[video or url](/path/to/video) 视频
@[pdf](/path/to/pdf) PDF
@[msoffice](/path/to/msoffice-file) Microsoft Office 文件(仅限公网地址)
@[bilibili](BV or AV) 哔哩哔哩视频
@[youtube](watch-id) YouTube 视频 (如果看不到就是网络问题，本站只提供解析，决不会提供翻墙服务)
@[user](uid) 提及用户

另外，markdown-it-media 也支持 vimeo、vine、prezi 平台的引用，虽然通常不会用到。关于如何使用 markdown-it-imsize 和 markdown-it-footnote 请自行查阅。markdown-it-merge-cells 会合并列表中相邻的相同元素。

为什么我的 LaTeX 不起作用？请按照 KaTeX 的规范。如$ x^2+y^2=z^2 $不能正常渲染，而$x^2+y^2=z^2$可以正常渲染。
